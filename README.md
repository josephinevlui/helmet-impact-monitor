# Helmet Impact Monitor

A wearable head impact indicator device for concussion risk monitoring in youth athletes.

## Hardware
- Arduino Nano 33 BLE Rev2
- ADXL375 High-g Accelerometer
- MPU9250 Gyroscope
- RGB LEDs for severity indication

## Project Structure
- firmware/ - Arduino embedded code
- software/ - Python Streamlit dashboard
- models/   - Injury risk models
- data/      - Impact data logs
- analysis/  - Notebooks and analysis scripts
- docs/      - Documentation and diagrams
- tests/     - Unit tests
